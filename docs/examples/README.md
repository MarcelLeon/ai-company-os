# Examples

这些 examples 是给开源用户看的第一眼体验入口:不是证明某个命令存在,而是展示
AI Company OS 如何把散落的 AI CLI 组织成一个可远程管理的项目团队。

## 主 Demo

- [Release Room](release-room.md):用 Telegram/Feishu 远程指挥一个 AI 团队完成小型
  开源 CLI 的 v0.2 发布。它串起 project office、team appointment、role proposal、
  shared memory、approval/audit、task trace、metrics 和 overnight handoff。

## 选择标准

一个 demo 必须同时满足:

- **现实工作**:像真实开发者会遇到的项目协作,不是为了展示命令而编排命令。
- **端到端**:有仓库、任务、团队、验收、失败恢复和交接,不是单点 prompt。
- **可验证**:用户可以按 playbook 运行,并能看到任务、审计、报告和交付物。
- **不夸大**:AICO 负责组织 AI 工作,不把底层 AI CLI 的生成能力包装成 AICO 自己的魔法。
